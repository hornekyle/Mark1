Mark 1 MD Code


================ metal units ==================
mass              = grams/mole
distance          = Angstroms
time              = picoseconds
energy            = eV
velocity          = Angstroms/picosecond
force             = eV/Angstrom
torque            = eV
temperature       = Kelvin
pressure          = bars
charge            = multiple of electron charge
dipole            = charge*Angstroms
density           = gram/cm^dim
dynamic viscosity = Poise
electric field    = volts/Angstrom
